Lesson 10 (HTML Tables)

Creating Tables

Tables are defined by using the table tag.
Tables are divided into table rows with the tr tag.
Table rows are divided into table columns (table data) with the td tag.

Note:
• Table data tags td act as data containers within the table. They can contain all sorts of HTML elements, such as text, images, lists, other tables, and so on.
• td tag is used to create columns in a row
• The border attribute is not supported in HTML5.
• Colspan attribute is used to expand a cell for two or more cells
• To make a cell span more than one row, use the rowspan attribute.
